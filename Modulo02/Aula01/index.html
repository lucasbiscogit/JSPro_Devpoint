O que é HTML

: Define o início e o fim do documento HTML. : Contém informações sobre o documento, como título, links para estilos, scripts, etc. : Define o título do documento, que é exibido na barra de título do navegador. <body>: Contém o conteúdo visível do documento. <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: São usadas para definir cabeçalhos de diferentes níveis. <p>: Define um parágrafo. <a>: Define um hiperlink. <img>: Insere uma imagem. <ul>: Define uma lista não ordenada. <ol>: Define uma lista ordenada. <li>: Define um item de lista, dentro de <ul> ou <ol>. <div>: Define uma divisão ou seção genérica em um documento. <span>: Define uma parte de texto para aplicação de estilo ou manipulação via JavaScript. <table>: Define uma tabela. <tr>: Define uma linha em uma tabela. <td>: Define uma célula de dados em uma tabela. <form>: Define um formulário para entrada de dados. <input>: Define um campo de entrada em um formulário. <textarea>: Define uma área de texto multilinha em um formulário. <button>: Define um botão clicável.
